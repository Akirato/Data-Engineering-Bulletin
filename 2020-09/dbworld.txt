The September, 2020 issue of the Data Engineering Bulletin is now
available electronically via the IEEE Computer Society Bulletin web
site http://tab.computer.org/tcde/bull_about.html. The ISSUES pages
are maintained both at the IEEE Computer Society
(http://sites.computer.org/debull/bull_issues.html) and at Microsoft
(ftp://ftp.research.microsoft.com/pub/debull/bull_issues.html).

This issue features M. Tamer Ozsu's opinion titled ``A Systematic View
of Data Science.'' It is a follow-up of Jeffrey Ullman's opinion ``The
battle for Data Science'' in the June 2020 issue, and it dives deeper
and raises critical questions regarding the formation of a new field.

Lei Chen served as the assoiciate editor for this issue, which
features 5 articles that address ``human-powered AI'' from several
very interesting angles.
